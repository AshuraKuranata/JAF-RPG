# JAF-RPG Project

The intent and purpose of this project is to build a full stack web application RPG game.

My goals for the project are as follows:

- Implement a frontend framework (React.js)
- Implement a backend framework (Express.js)
- Implement mulitple databases (PostgreSQL) with models and tables that integrate with each other 
- Implement a frontend framework (Tailwind) successfully
- Utilize Tailwind styles to create a visually appealing game
- Implement proper user authentication protocols to protect user data (Cookie & JWTs)
- Learn and implement testing
- Learn and implement documentation
- Learn and implement reliable cloud deployment for web application to stay live
- Game actions should utilize stateful management
- Implement variety of typical front end features for web application to build a unique user experience

Some of the game's features will include:

- Character creation system
- Battle system
- Leveling system
- Inventory system
- Item system
- Magic system
- User authentication system
- High score system
- Cookie-based user data storage or JWT authentication
- Cloud deployment
- Documentation
- Testing
