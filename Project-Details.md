# JAF-RPG Project Details

Just Another Fantasy RPG (JAF-RPG)

## Expectations

Develop an RPG browser game with the following features:

Features:
- Character creation system
- Battle system
- Leveling system
- Inventory system
- Item system
- Magic system
- User authentication system
- High score system
- Cookie-based user data storage or JWT authentication
- Cloud deployment
- Documentation
- Testing

## Technologies

This project will be built using the following technologies:

Frontend:
- HTML/CSS (Tailwind)
- JavaScript
- Node.js

Frontend Framework:
- React.js

Backend Framework:
- Express.js
- Python (Django)

Backend:
- PostgreSQL
- GitHub

IDE:
- VS Code
